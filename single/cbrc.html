CBRC

For this project I took over an older project started by a former intern at the Emerge Studio. The California Bird Record Committee —CBRC— asked our studio to redesign their website but leave them the ability to edit and maintain it with minimal codinq requirements.

Role

Project Manager

Visual Strategy

The main focus for this project was to develop a highly intuitive user experience for people who were not used to navigating websites on an ever yday basis. To achieve this, we developed an interface design with large text and bold buttons. We simplified the navigation and redefined the architecture.

Client

California Bird Records Committee (2016)

Challenges

The original website made use of outdated technology and practices, and the main goal of this project was to redesign the website to be responsive and up to modern standards.The client was reticent to shift the website onto an entirely new framework, and a significant portion of my work involved translating the changes into an easily digestible format for the client. For this project I had to manage a ten-person team with widely varied skillsets.

Tools

Balsamiq, Sketch, HTML/CSS, Javascript
