Type Layouts

As an in-class exercise, I was asked to design different layouts for a weekly study guide over the duration of the semester.

Role

Designer

Visual Strategy

The study guides were about readings, typesetting and specific type families. Each layout was reinforced the reading using only type elements.

Client

Student Project (2016)

Challenges

This exercise went on for ten weeks and I had to design a unique layout for all of them. At the beginning it was a challenge to find new ideas each time but in the end I felt more comfortable experimenting with shapes and format.
